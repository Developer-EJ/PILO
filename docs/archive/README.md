# PILO Archived Docs

이 폴더는 과거 기획서, 분업안, 검토 리포트, legacy 협업 문서를 보관한다.

규칙:

1. 이 폴더의 문서는 구현 기준이 아니다.
2. AI agent 작업 지시에는 이 폴더의 문서를 source of truth로 넣지 않는다.
3. archive 문서에서 되살릴 내용은 먼저 `docs/mvp-scope-v1.md`, `docs/domain-boundary-v1.md`, `docs/api-contract-v1.md`, `docs/db/mvp-db-schema-v1.md` 중 적절한 기준 문서에 반영한다.
4. 결정 배경을 확인할 때만 읽는다.
